Used to isolate whether Chromium on file:// treats module scripts differently for CORS purposes — which would explain why probe.html (classic) reaches Ask Sage from your DHA workstation but the SPA bundle (module) does not.

Origin: . Open this file two ways and compare results: (1) double-click to open from file://; (2) serve it locally (e.g. python -m http.server in this folder, then visit http://localhost:8000/probe.html). Some browsers treat file:// as the opaque origin null, which CORS-allowed APIs may still reject.
